# Data

The notebook uses game-level NBA player statistics from `PlayerStatistics.csv`.

Place the dataset in this directory before running the notebook:

```text
data/PlayerStatistics.csv
```

The expected data includes player and game identifiers, game dates, teams and opponents, minutes, points, shooting attempts and percentages, rebounds, assists, steals, blocks, fouls, turnovers, and home/away status.

The CSV is not committed because it is large. The repository's `.gitignore` keeps local dataset copies out of version control.
